# Cyber Chef

## What it is 
- CyberChef is a web-based tool used for encoding, decoding, encrypting, decrypting, and transforming data.

## Why it matters in Cybersecurity 
- Widely used by SOC analysts and forensic investigators to quickly analyze suspicious data
- Helps decode artifacts from logs, malware samples, phishing emails, and network traffic

## What i practiced
- Extracting indicators (IP addresses, emails, URLs) from raw text
- Converting and normalizing date/time formats
- Decoding and transforming data formats (Base64, Base85, etc.)

## Quick References
- Operations: Extract URLs, Extract IP addresses, From Base64, From Base85
- Common workflow: Paste data → build recipe → export output

## Key Takeaways
- Fast and accurate data transformation is critical in SOC work to extract and validate indicators efficiently
