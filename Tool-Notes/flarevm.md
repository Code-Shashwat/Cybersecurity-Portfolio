# Flare VM

## What it is 
- FLARE VM (Forensics, Logic Analysis, and Reverse Engineering) is a Windows-based security toolkit that provides a curated set of tools for malware analysis, reverse engineering, forensics, and incident response.

## Why it matters in Cybersecurity 
- Provides a ready-to-use lab environment for malware analysis and investigation
- Helps analysts safely examine suspicious files using industry-standard tools

## What i practiced
- Exploring key tools included in FLARE VM such as Procmon, PE Studio, Wireshark, CFF Explorer, and Process Explorer
- Performing basic analysis of suspicious/malware samples in a controlled environment

## Quick References
- Use FLARE VM inside a virtual machine (VM) for safe analysis
- Common tools: Procmon, Process Explorer, PE Studio, Wireshark, CFF Explorer

## Key Takeaways
- Malware analysis requires multiple tools, and having a dedicated lab environment is essential
- Suspicious files should always be handled carefully in an isolated VM to avoid infecting a real system
