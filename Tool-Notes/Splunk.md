# Splunk

## What it is
Splunk is a SIEM and log analysis platform used to search, monitor, and analyze machine data using SPL (Search Processing Language).

## Why it matters in cybersecurity
- Widely used in SOC environments for log analysis and alerting  
- Enables fast searching and correlation of large datasets  
- Supports detection use cases and dashboard creation  

## What I practiced
- Searching logs using basic SPL queries  
- Filtering events based on fields (IP, user, event type)  
- Identifying suspicious login attempts  
- Understanding dashboards and alerts  

## Quick Reference
- Basic search: `index=*`
- Filter by field: `index=windows user=admin`
- Common commands: `search`, `stats`, `table`

## Key Takeaways
- Efficient searching is key to fast investigation  
- Understanding log structure improves detection capability
