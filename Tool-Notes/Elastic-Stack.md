# Elastic Stack

## What it is
Elastic Stack (ELK) is a collection of tools (Elasticsearch, Logstash, Kibana) used for log collection, storage, search, and visualization.

## Why it matters in cybersecurity
- Provides powerful log search and visualization capabilities  
- Used for monitoring, alerting, and threat detection  
- Common alternative to traditional SIEM tools  

## What I practiced
- Exploring logs in Kibana  
- Filtering and searching events  
- Visualizing data using dashboards  
- Understanding log ingestion flow  

## Quick Reference
- Components:
  - Elasticsearch → storage & search  
  - Logstash → log processing  
  - Kibana → visualization  

## Key Takeaways
- Visualization helps quickly identify anomalies  
- Understanding data flow is important for troubleshooting
