# SIEM Tools (Splunk/Elastic Stack)

## What it is
A SIEM (Security Information and Event Management) system collects, normalizes, and analyzes logs from multiple sources (endpoints, servers, network devices) to detect and investigate security events.


## Why it matters in cybersecurity
- Centralized visibility across the environment  
- Enables alerting and correlation of suspicious activities  
- Core tool used by SOC analysts for monitoring and investigation  

## What I practiced
- Searching and filtering logs  
- Identifying suspicious login activity  
- Correlating events across multiple log sources  
- Understanding alert generation and triage workflow  

## Quick Reference
- Log sources: Windows, Linux, Network, Applications  
- Key concepts: Events, Alerts, Correlation, Dashboards  

## Key Takeaways
- SIEM is the primary tool for SOC investigations  
- Effective log analysis is critical for detecting threats
