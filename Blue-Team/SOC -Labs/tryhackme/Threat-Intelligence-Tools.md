# Threat Intelligence Tools (Virus Total,Malware Bazaar,Nslookup, Ipinfo)

## What it is
Threat intelligence tools provide information about known malicious indicators such as IPs, domains, URLs, and file hashes.

## Why it matters in cybersecurity
- Helps validate whether an indicator is malicious  
- Supports faster investigation and decision-making  
- Improves detection accuracy and reduces false positives  

## What I practiced
- Checking IP/domain reputation  
- Validating file hashes  
- Using threat intel to enrich investigation data  

## Quick Reference
- Common indicators: IP, Domain, URL, Hash  
- Usage: Input indicator → Check reputation → Correlate with logs  

## Key Takeaways
- Threat intelligence is used to support, not replace, investigation  
- Always correlate intel with actual logs before making conclusions
