# Try Hack me - SOC Level 1

## Purpose 
This document summarizes the concept and tools learned from Try Hack me SOC level 1 learning path.

---
## Overview
The SOC Level 1 path focuses on building practical skills required for a Security Operations Center (SOC) role. It covers alert triage, log analysis, phishing investigation, network traffic analysis, and basic incident response concepts.

This path helped me understand how security analysts investigate alerts and identify suspicious activity using logs and tools.

---

## Key Topic Covered 
- SOC workflow and alert triage process  
- Understanding logs from Windows, Linux, and network sources  
- Phishing analysis (email headers, links, attachments)
- Core SOC Solution (EDR, SIEM, SOAR)
- Cyber Defence Frameworks ( Pyramid of pain, Cyber Kill Chain, Unified kill chain)
- Network traffic analysis (DNS, HTTP, TLS basics)
- Network Security Monitoring (IPS,IDS)
- Indicators of Compromise (IOCs) and threat intelligence basics
- Malware Concepts
- MITRE ATT&CK mapping for attacker techniques  
- Basics of incident response and investigation workflow  

---

## Practical Skill Learned
- Investigating alerts and identifying false positives vs real threats  
- Analyzing Windows and Linux logs for suspicious activity  
- Performing phishing investigations and identifying malicious indicators  
- Using network analysis techniques to detect suspicious traffic  
- Extracting and validating IOCs (IP, domain, hash)  
- Documenting findings in a structured investigation format  
---

## Tools learned 
- EDR
- SIEM (Splunk, Elastic Stack)
- Linux CLI and log analysis  
- Windows Event Viewer / Sysmon
- Threat intelleligence tools (Virus total, Malware bazaar, Hybrid Analysis,Sodan,Cisco Talos,Nslookup)
- IPS/IDS (Snort)
